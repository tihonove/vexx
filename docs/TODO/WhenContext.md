# Система контекста (when)

## Цель

Реализовать систему контекстных условий по аналогии с VS Code `when` clause — определяет, когда команда/кейбиндинг/пункт меню активен.

## Подзадачи

### [x] 1. Спроектировать и реализовать ContextKeyService

Хранилище ключ→значение, типизированное по `ContextKey`. Eval-based вычисление when-выражений (`&&`, `||`, `!`).

### [ ] 2. Парсинг и вычисление when-выражений

Сейчас используется `new Function(...)` с подстановкой переменных. В будущем можно заменить на полноценный парсер для безопасности и расширяемости.

### [x] 3. Интегрировать с KeybindingRegistry и меню

`KeybindingRegistry.register()` принимает `when?` строку, `resolve()` проверяет условие через `ContextKeyService.evaluate()`.
